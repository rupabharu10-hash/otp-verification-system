# OTP Verification System

A Python-based OTP authentication system that generates a secure 6-digit OTP,
simulates email delivery, and verifies user input with retry limits.

## Features
- Secure 6-digit OTP generation
- Hidden OTP input using getpass
- Retry limit to prevent brute-force attempts
- Modular and easy-to-read code

## Technologies Used
- Python
- random
- time
- getpass

## How to Run
```bash
python otp_verification.py
